# Frame Spirit - Telegram Mini App

## Описание
Telegram Mini App для продажи услуг по сборке, чистке и обслуживанию компьютеров.

## Особенности
- Современный дизайн с анимациями
- Адаптивная верстка
- PWA поддержка
- Локальное хранение данных
- Интеграция с Telegram Web App API
